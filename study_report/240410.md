### 泛型
1. 理解：同一个函数，支持不同的类型

2. 用法：
   ```
    struct Data<T>  //struct 结构体名称<T>
    {  
       value:T,     //元素：T
    }
    
    fn main() {
        let t:Data<i32> = Data{value:100};
        println!("值:{} ",t.value);//输出 值:100
        let t:Data<f64> = Data{value:66.00};
        println!("值:{} ",t.value);//输出 值:66
    }
   ```
