
### 错误处理 续
- 不可恢复错误:`panic!()` 
  
  ```
  fn main()
  {
    panic!("出错啦");
    println!("Hello Rust"); // 不可能执行的语句
  }
  //输出
  thread 'main' panicked at '出错啦', src/main.rs:2:5
  
  
  let v = vec!["哈哈","呃呃","嘻嘻"];
  v[5]; // 因为超出了数组的长度，所以会触发不可恢复错误
  ```
